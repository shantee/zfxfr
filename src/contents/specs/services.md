# Mes services

## Dépannage PC "Ma machine marche mal!"

- Diagnostic & dépannage logiciel (erreurs, lenteurs, plantages…)
- Nettoyage **logiciel** + optimisation (démarrage, programmes inutiles, mises à jour)
- Aide en cas de **mot de passe oublié** (récupération / réinitialisation quand c’est possible)
- Récupération de données **selon l’état du disque** (fichiers supprimés, inaccessibles, disque capricieux)
- Sauvegardes simples et fiables (disque dur externe, et/ou cloud si besoin)
- Transfert vers un nouveau PC (documents, photos, favoris, emails…)
- Installation / réinstallation Windows (propre + réglages de base + pilotes)
- Installation / dual-boot Linux (Ubuntu, Mint…) + prise en main

## Internet / Wi-Fi / imprimante 

- Diagnostic et optimisation Wi-Fi / box / réseau domestique (répéteur, couverture, stabilité)
- Installation / configuration imprimante & scanner (USB ou Wi-Fi)
- Partage imprimante / fichiers entre plusieurs appareils (PC, téléphone, tablette)

## Téléphones & tablettes

- Prise en main Android (réglages, stockage, applis utiles)
- Sauvegarde / transfert vers un nouveau téléphone (photos, contacts, WhatsApp, etc.)
- Nettoyage + remise à zéro (sauvegarde puis réinitialisation propre)

## Cours et accompagnement (assistance , coaching , formations) 

- Cours “bases de l’informatique” (Windows, fichiers, Internet, emails)
- Coaching numérique seniors (à votre rythme, sans stress, sans jargon)
- Organisation des photos et documents (classement, doublons, sauvegarde)
- Installation + configuration d’un gestionnaire de mots de passe (Bitwarden, etc.) + mini cours.
- Sécurité de base (mots de passe, arnaques/phishing, mises à jour, bonnes habitudes)

## Développement web & outils sur mesure

- Création de site vitrine / blog (WordPress / Astro…)
- Optimisation, maintenance et petits correctifs sur site existant
- Automatisation de tâches (scripts, petits outils)
- Développement de scripts et logiciels sur mesure
- WebGL / 3D (démo, présentation produit, animation)

## Divers (selon besoin)
